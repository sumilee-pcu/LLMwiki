# 프롬프트·LLM 활용 도서 목록 2026-05

## 자료 개요

개인 보유 전자책 라이브러리(약 461개 PDF) 중 [[Recommended Prompt Library|분야별 추천 프롬프트 라이브러리]]의 8개 분야와 [[LLM Usage Economics|LLM 사용 경제학]]의 Tier 모델에 직접 연결되는 도서를 선별한 카드다. 향후 실제 패턴·프롬프트를 추출해 라이브러리에 반영하기 위한 참조 인벤토리 역할을 한다.

## 선별 기준

- 프롬프트 패턴·예제가 본문 중심을 차지하는 도서
- 멀티 모델(ChatGPT, Gemini, Claude, Perplexity 등) 비교가 가능한 도서
- 에이전트·자동화·MCP·RAG 등 프로그래매틱 사용 패턴을 다루는 도서
- 강의·논문·콘텐츠 제작 등 분야별 적용 사례를 담은 도서

이론·게임·네트워크·통계·하드웨어 등 본 라이브러리의 직접 활용 범위 밖 도서는 본 카드에서 제외한다.

## 우선 추출 후보

위키 콘텐츠에 가장 빠르게 반영 효과가 큰 도서다.

1. `프롬프트를만드는프롬프트gptsnGems_opt.pdf` — 메타 프롬프트 패턴을 라이브러리 각 분야에 직접 이식 가능
2. `챗GPT퍼플렉시티클로드코파일럿제미나이다잘함.pdf` — 멀티 모델 비교 표를 라이브러리 "모델 선택 가이드"에 보강
3. `이게되네챗GPT71제.pdf` — 71개 케이스에서 분야별 매트릭스 보강
4. `컨텍스트엔지니어링으로완성하는AI에이전트.pdf` — 하네스 엔지니어링과 LLM 사용 경제학 양쪽에 인용
5. `바이브코더를위한최소한의AIIT지식.pdf` — 바이브 코딩 활용 사례 페이지의 진입 가이드 보강
