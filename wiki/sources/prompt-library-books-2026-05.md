# 프롬프트·LLM 활용 도서 목록 2026-05

## 자료 개요

시중 출판 교재 중 [[Recommended Prompt Library|분야별 추천 프롬프트 라이브러리]]의 8개 분야와 [[LLM Usage Economics|LLM 사용 경제학]]의 Tier 모델에 직접 연결되는 도서를 선별한 카드다. 향후 실제 패턴·프롬프트를 추출해 라이브러리에 반영하기 위한 참조 인벤토리 역할을 한다.

## 선별 기준

- 프롬프트 패턴·예제가 본문 중심을 차지하는 도서
- 멀티 모델(ChatGPT, Gemini, Claude, Perplexity 등) 비교가 가능한 도서
- 에이전트·자동화·MCP·RAG 등 프로그래매틱 사용 패턴을 다루는 도서
- 강의·논문·콘텐츠 제작 등 분야별 적용 사례를 담은 도서

이론·게임·네트워크·통계·하드웨어 등 본 카드의 직접 활용 범위 밖 도서는 제외한다.
