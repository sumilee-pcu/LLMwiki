# 유용한 GitHub 링크 목록 2026-05-18

## 자료 개요

사용자가 제공한 유용한 GitHub 저장소와 프로젝트 페이지 목록이다. 한국 법률 MCP, 보이스 에이전트, 화면 녹화 앱, 카카오톡 MCP/CLI, 이미지 기반 3D 생성, 블로그 기반 숏폼 생성 플러그인이 포함되어 있다.

| 항목 | 출처 링크 | 확인 메모 |
